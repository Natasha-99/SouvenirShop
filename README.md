## Магазин сувениров

Информационная база предназначена для автоматизации процессов управления закупками и продажами в магазине сувениров. 

**ИБ магазина сувениров выполняет следующие функции:**
* Ведение каталога товаров;
* Управление запасами товаров;
* Оформление заказов;
* Осуществление учета продаж;
* Управление информацией о контрагентах;
* Внесение в базу новых должностей и подразделений.
